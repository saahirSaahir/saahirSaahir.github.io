Experience

Argonne National Laboratory

Robotics Engineer Intern

Working with Dr. Young-Soo Park of the Applied Materials Division, I developed a hardware-in-the-loop simulation to be able to do affordable prototyping and testing of various tooling tasks using a single robot simultaneously with virtual simulations. This makes testing more efficient and the actual recreation of movement is very accurate. This project involved using V-REP, ROS, Python and Lua.

Argonne National Laboratory

Electrical Engineer Intern

This summer I worked with Dr. Andrew Chuang of the X-Ray Science Division, I created an image processing procedure using MATLAB, Python and ImageJ to process thousands of x-ray tomography data of Ti-6Al-4V powder obtained using the Advanced Photon Source (APS) for additive manufacturing and identifying porosity and impurities. The process reached 96% accuracy and also 3D reconstruction of these pores was done using MATLAB.

ADN Telecom Limited

Radio/Optics Team Engineer

I got the chance to work with the radio team in setting up radio devices to provide cell network to various places around Dhaka city and helped build a point-of-presence in Mothijheel area. This involved learning about SDH multiplexers and CISCO commands associated with setting up these points. I also worked with the optics team in setting up broadband connections and interacting with clients.
